## Quelques outils utiles...

Définition de l'éditeur de texte

```bash
git config --global core.editor vi
```

Outil de diff

```bash
git config --global merge.tool vimdiff
```

Vérification de la configuration

```bash
git config --list
# Ou
git config {key}
```

Aide

```bash
git {verb} --help
```
