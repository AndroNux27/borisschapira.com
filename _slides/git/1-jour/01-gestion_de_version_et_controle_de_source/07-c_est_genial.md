## **git**, mais pourquoi ?

* Chaque version de fichier est sauvegardée (pas les deltas)
* Rapide comme l'éclair
* Fonctionne en local
* Quasiment impossible de perdre des données

<aside class="notes">
  SVN, CVS, Perforce, Mercurial… enregistrent les changements sur les fichiers.
</aside>
