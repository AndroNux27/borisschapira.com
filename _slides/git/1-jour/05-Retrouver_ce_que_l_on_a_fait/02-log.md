## Parcourir l’historique

```bash
git log
git log <branch>
git log --author=Toto
git log --grep=keyword -i -E
git log --graph
git log --format=oneline
git log --since="yesterday"
git log --since="1 hour ago"
git lg # uniquement si l'alias est défini
```
